
# Widget Swapping




## Run Locally

Clone the project

```bash
  git clone https://github.com/R-ohit-B-isht/flutter-WidgetSwapping.git
```

Open the project directory in Vs Code / android Studios

```bash
  cd my-project
```

Just run

```bash
  flutter run -d chrome
```
